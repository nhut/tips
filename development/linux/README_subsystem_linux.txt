## Install & Uninstall Subsystem Linux (Ubuntu) for Windows
https://www.howtogeek.com/261188/how-to-uninstall-or-reinstall-windows-10s-ubuntu-bash-shell/

### Install Subsystem Linux (root as default user)
> lxrun /install /y

### Uninstall Subsystem Linux (keep home folders)
> lxrun /uninstall

### Uninstall Subsystem Linux (all)
> lxrun /uninstall /full
